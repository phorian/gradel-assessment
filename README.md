# Description
This is a sample Exercise for gradel
#Installation
$ npm install
# Run the app
   Start
  $ npm run start
   Development
  $ npm run dev
    Build
  $ npm run build
# Test
   To run all tests:
  $ npm test

    To run only unit tests:
  $ npm run test:unit

    To run only integration tests:
  $npm run test:integration

    To run tests with coverage report:
  $ npm run test:coverage
